# 153. 寻找旋转排序数组中的最小值

假设按照升序排序的数组在预先未知的某个点上进行了旋转。

( 例如，数组 `[0,1,2,4,5,6,7]` 可能变为 `[4,5,6,7,0,1,2]` )。

请找出其中最小的元素。

你可以假设数组中不存在重复元素。

**示例 1:**

```
输入: [3,4,5,1,2]
输出: 1
```

**示例 2:**

```
输入: [4,5,6,7,0,1,2]
输出: 0
```



***

## CPP

```cpp
/*
执行用时：8 ms
内存消耗：10.2 MB
*/
    class Solution {
public:
    int findMin(vector<int>& nums) {
        sort(nums.begin(), nums.end());
        return nums[0];

    }
};
/*
Time:O(nlogn) 排序时间
Space:O(1)
*/
```



```cpp
//执行用时为 0 ms 的范例
class Solution {
public:
    int findMin(vector<int>& nums) {
        int n = nums.size();
        int res = nums[0];
        for (int i = 1; i < n; i++)
        {
            if (nums[i] < nums[i -1])
            {
                res = nums[i];
                break;
            }
        }
        return res;
    }
};
```
